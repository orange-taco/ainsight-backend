- main : 제품으로 출시될 수 있는 브랜치
- develop : 다음 출시 버전을 개발하는 브랜치(master에서 분기)
- feature : 기능을 개발하는 브랜치(develop에서 분기)
- release : 이번 출시 버전을 준비하는 브랜치(QA / develop에서 분기)
- hotfix : 출시 버전에서 발생한 버그를 수정 하는 브랜치(master에서 분기)
